# 대결 포인트 1원 단위 표시 — 2026-06-17 (yeoneelee)

파일: `frontend/src/screens/battle/ActiveBattle.tsx`

## 목표

대결 화면에서 포인트를 "100만P" 같은 축약형이 아닌 "1,000,000P" 형태로 1원 단위까지 정확하게 표시.

## 구현

- `fmtP` 함수에서 억/만 단위 축약 로직 제거
- `n.toLocaleString()` 으로 통일 (천 단위 쉼표 구분)
- 적용 범위: 내 순위 카드 총자산, 순위 탭 총자산, 내 투자 탭 현금/주식평가액, 거래 탭 총금액/가용현금
