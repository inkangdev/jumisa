# 대결 레이스 UI 보완 — 2026-06-18 (yeoneelee)

파일: `frontend/src/screens/battle/ActiveBattle.tsx`

## 문제

이전 구현(`2026-06-17-yeoneelee-대결레이스UI.md`)은 각 참가자가 고정된 별도 레인에서 아바타만 좌우로 이동하는 방식이었다. 순위가 역전돼도 행 순서가 바뀌지 않아 "추월" 순간이 시각적으로 전혀 드러나지 않았다.

## 목표

선거개표방송처럼 순위 역전 시 행 전체가 수직으로 이동해 실시간 추격·추월 장면이 UI에 담기도록 개선.

## 변경 내용

- 각 행을 `position: absolute` + `top: index * SLOT(46px)` 으로 배치
- `transition: "top 1.2s cubic-bezier(0.34,1.56,0.64,1)"` 적용 → 10초 폴링으로 `ranking` 배열 갱신 시 순위가 바뀐 참가자 행이 수직으로 이동하며 서로 스치는 추월 연출
- 바를 공통 좌측 기준선(left=0)에서 오른쪽으로 성장하는 구조로 변경 (아바타 제거, 바 길이로 순위 차이 직관적 표현)
- 수익률을 각 행 우측에 `+X.XX%` 형태로 표시 (초록/빨강 색상 구분)
- 닉네임은 바 내부 좌측에 오버레이
- 내 바: accent 파랑 / 남 바: 수익 초록, 손실 빨강
