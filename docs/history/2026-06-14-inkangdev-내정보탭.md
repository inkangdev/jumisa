# 2026-06-14 · 하단 네비 '더보기' → '내 정보' 탭 교체

브랜치: `feature/profile-tab` (main 기준)

메인 셸(틀) 후속. 하단 4번째 탭을 `더보기(☰)` 대신 **`내 정보`** 로 바꾸고, 그 탭 아이콘은 **로그인 사용자의 프로필 동물(아바타)** 로 동적 표시한다. 기능정의서 2장 "회원 정보 페이지" 자리.

## 한 일

- `layout/nav.ts` — `AppTab` `"more"` → `"profile"`, 라벨 `더보기` → `내 정보`. icon은 fallback(🐂), 실제론 아바타로 대체.
- `layout/BottomNav.tsx` — `avatar` prop 추가. `profile` 탭 아이콘을 `avatar ?? n.icon`(사용자 아바타)으로 렌더.
- `layout/AppShell.tsx` — `BottomNav`에 `avatar={user.avatar}` 전달. 자리표시의 `profile` 탭: 큰 아이콘도 아바타로, 프로필 블록은 닉네임 + 로그아웃(아바타 중복 제거). `tab === "more"` → `"profile"`.

## 검증

- 로컬 빌드 불가(node 없음) → 코드 리뷰로 타입 확인. 머지 후 Render 빌드 + 라이브 번들 해시 변경으로 반영 확인.
- 수동: 로그인 → 하단 4번째 탭이 내 아바타(동물) 아이콘 + "내 정보" → 탭 진입 시 아바타/닉네임/로그아웃.

## 후속

- 내 정보 탭 실제 화면(프로필 편집, 보유 종목, 모의투자 통계 등 — 기능정의서 2장).
