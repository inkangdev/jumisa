# 대결 UI 개선 — 포인트 슬라이더 & 수량 입력

## 작업 내용

### 1. 대결방 생성 시작 포인트 개선 (`CreateRoom.tsx`)

**변경 전**: 50만 / 100만 / 300만 고정 버튼 3개

**변경 후**:
- 슬라이더(range input): 100만 ~ 1억5천만, 100만 단위 스텝
- 현재 선택값 크게 표시 (예: `500만 P`)
- 빠른 프리셋 버튼: 100만 / 300만 / 500만 / 1000만 / 3000만 / 5000만 / 1억 / 1억5천
- `fmtPoint()` 헬퍼로 1억 이상은 `1억`, `1억 5천만` 식으로 표기

### 2. 거래 수량 입력 개선 (`ActiveBattle.tsx` — TradeTab)

**변경 전**: − / 숫자표시(클릭불가) / + 버튼만 있어 많은 수량 입력 불편

**변경 후**:
- 수량 표시를 직접 입력 가능한 `<input type="number">` 로 변경
- 비율 버튼 4개 추가: `25%` / `50%` / `75%` / `최대(매수)` or `전량(매도)`
  - 매수 시: 가용현금 ÷ 현재가 기준 최대 수량에서 비율 계산
  - 매도 시: 보유 수량 기준 비율 계산
- 가용현금 표시 옆에 `(최대 N주)` 안내 추가
