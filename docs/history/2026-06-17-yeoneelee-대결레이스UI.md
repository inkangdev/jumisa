# 대결 순위 탭 경주마 레이스 UI — 2026-06-17 (yeoneelee)

파일: `frontend/src/screens/battle/ActiveBattle.tsx`

## 목표

순위 탭 상단에 경주마처럼 각 아바타가 수익률 기준으로 위치를 잡고 순위 변동 시 역동적으로 이동하는 레이스 UI 추가. 수익률 숫자 표시는 아래 블록이 담당하고, 레이스 트랙은 상대적 순위를 시각적으로 강조.

## 구현

- `RaceTrack` 컴포넌트 신규 추가
- 참가자별 가로 레인, 수익률을 12%~88% 범위로 정규화해 아바타 위치 결정
- 전원 동률(대결 초반 0%)이면 모두 중앙(50%) 배치
- 10초 폴링으로 데이터 갱신 시 아바타·진행 바가 `cubic-bezier(0.34,1.56,0.64,1)` 스프링으로 슬라이드
- 내 아바타에 파란 glow(`drop-shadow`) 강조
- 진행 바는 `overflow: hidden` 내부 레이어로 처리, 아바타는 별도 레이어로 트랙 경계에서 잘리지 않도록 분리
- `RankTab`에서 `RaceTrack` → 기존 `AnimatedRankRow` 블록 순서로 배치
