# Jumisa 기능 정의서 v1.0

## 1. 저평가 주식 스크리너

- 국내/미국 탭 구분 (미국 포함 여부 추후 확정)
- 섹터/업종 필터
- 저평가 점수 산출 및 랭킹 (점수 배분 기준 별도 정의 필요)
- PER/PBR/EV/EBITDA 등 주식 용어 → 입문자도 이해할 수 있는 쉬운 용어로 병행 표기

### 데이터 소스

- 국내 주가 (1시간 단위) → 한국투자증권 Open API
- 국내 재무지표 → 금융위원회 공공데이터포털
- 미국 주식 → FMP API (확정 시)

### 뉴스 / AI 분석

- 네이버 금융 RSS로 종목별 뉴스 수집
- Claude API로 뉴스 감성 분석 및 종목 현황 요약 제공
- 미국 종목 뉴스 → FMP 뉴스 엔드포인트

## 2. 로그인 / 회원

### 로그인

- 아이디/비밀번호 or 간편 소셜 로그인 (확정 필요)
- 비밀번호 변경 / 회원 탈퇴

### 회원 정보 페이지

- 별명 / 프로필 사진
- 보유 주식 관리 (사용자 수동 입력, 종목 코드 입력 시 현재가 자동 연동)
- 모의투자 포인트 / 보유 종목 / 수익률 통계

## 3. 공통

- 관심 종목 등록 및 가격 변동 알림
- 저평가 랭킹 히스토리 (순위 추이 확인)
- 공지사항 / FAQ 페이지

## 4. 모의투자

### 포인트

- 로그인 시 초기 포인트 지급 (금액 확정 필요, 1포인트 = 1원)
- 포인트 소진 시 재충전 정책 확정 필요 (재로그인 지급? 주기 지급?)

### 매수 / 매도

- 기준가 → 1시간 단위 적재 주가 기준
- 시장가 즉시 체결 방식
- 해외 종목 → 매수 시점 환율 적용 (기준 확정 필요)
